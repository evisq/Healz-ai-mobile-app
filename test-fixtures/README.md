# Синтетические тестовые документы

Все файлы в этой папке созданы специально для разработки. Они не содержат
данных реального пациента и не являются медицинскими рекомендациями.

## Файлы

- `synthetic-lab-report.pdf` — двухстраничный PDF с текстовым слоем, таблицами и
  мелкими значениями.
- `synthetic-lab-report.png` — крупный PNG, похожий на скриншот документа.
- `synthetic-lab-photo.jpg` — JPEG высокого разрешения для проверки сжатия
  фотографий.
- `sources/synthetic-lab-report.html` — исходник для воспроизводимой генерации.

## Что проверять

- файл открывается после подготовки;
- мелкие значения читаются при увеличении 200–300%;
- количество страниц PDF не меняется;
- текст PDF можно выделить;
- обработанная копия используется только при уменьшении размера.

Запрещено заменять эти fixtures настоящими анализами или документами
пользователей.
